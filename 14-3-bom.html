setTimeout()

The global setTimeout() method sets a timer which executes a function or specified piece of code once the timer expires.

Syntax

var timeoutID = setTimeout(function[, delay, arg1, arg2, ...]);
var timeoutID = setTimeout(function[, delay]);
var timeoutID = setTimeout(code[, delay]); " not recommended"

Parameters

function

A function to be executed after the timer expires.

The time delay (Optional)

in milliseconds that the timer should wait before the specified function or code is executed. If this parameter is omitted, a value of 0 is used, meaning execute "immediately",

A function to be executed after the timer expires.
